Make Your Own Web Server

The task for this assignment was to make a node.js web server from "scratch" i.e. no frameworks such as express.

The server has to handle different file types and serve up error pages when there is something that is not handled or another type of error occurs.

Please not all test pages were created by my teacher Allan McDonald

Try some pages to test out what the server handles:

Searches in XML file and returns the first entry that matches the query.
Searches a JSON file and returns the entries that match the query.

A text file.

A form that posts.
A page that doesn't exist.
